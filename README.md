## 📖 Overview

**Daily AI Agent** is a fully automated GitHub bot that runs every day at **00:00 UTC** via GitHub Actions. It generates a daily markdown log and updates a JSON contribution tracker — keeping your GitHub contribution graph active, every single day.

Each daily run produces:
- 💬 A **motivational quote** from tech/industry legends
- 💡 A **tech tip** (Python, Git, JavaScript, SQL, AI/ML, Security, etc.)
- 🧠 An **AI fact of the day**
- 🎯 A **learning goal** to stay sharp
- 📊 An updated **contribution tracker**

## ⚙️ How It Works

1. **GitHub Actions** triggers the workflow every day at midnight UTC (or manually via `workflow_dispatch`).
2. The workflow checks out the repo, sets up Python 3.11, and installs dependencies.
3. **`agent.py`** runs and:
   - Picks a random motivational quote, tech tip, AI fact, and learning goal.
   - Writes a daily markdown log to `logs/YYYY-MM-DD.md`.
   - Updates `data/contribution_tracker.json` with the current run info.
   - Commits and pushes both files directly via the GitHub API.
4. A **job summary** is written to the Actions run for quick review.

## 🚀 Getting Started

### Fork & Enable

1. **Fork** this repository.
2. Go to **Settings → Actions → General** and ensure Actions are enabled.
3. The workflow will run automatically at 00:00 UTC every day.
4. To trigger it manually: go to **Actions → Daily AI Agent → Run workflow**.

### No secrets needed

The workflow uses the built-in `GITHUB_TOKEN` — no extra secrets required.

## 📊 Contribution Tracker

The file `data/contribution_tracker.json` tracks every run:

```json
{
  "total_runs": 3,
  "start_date": "2026-06-22",
  "last_run": "2026-06-22",
  "logs": [
    { "date": "2026-06-22", "run": 1 },
    { "date": "2026-06-22", "run": 2 },
    { "date": "2026-06-22", "run": 3 }
  ]
}
```

Up to the last **90 entries** are kept to avoid unbounded growth.
